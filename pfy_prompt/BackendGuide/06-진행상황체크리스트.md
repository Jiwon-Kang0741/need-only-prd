## ❓ 자주 발생하는 이슈

### 1. @ServiceId 인식 안됨
- **원인**: Framework 모듈 의존성 누락
- **해결**: tomms-fw-base 의존성 확인

### 2. MyBatis Mapper 파일을 못 찾음
- **원인**: Mapper XML 위치 오류
- **해결**: `src/main/resources/hsc/tomms/web/{module}/{screen}/mybatis/mappers/` 경로 확인

### 3. API 404 에러
- **원인**: @ServiceId 경로 오류
- **해결**: `/online/mvcJson/{ServiceId}` 형식 확인

### 4. DTO 필드 매핑 안됨
- **원인**: 필드명과 컬럼 alias 불일치
- **해결**: Mapper XML에서 `COLUMN_NAME as columnName` alias 확인

### 5. 트랜잭션 롤백 안됨
- **원인**: @Transactional 누락
- **해결**: CUD 메서드에 @Transactional 추가
