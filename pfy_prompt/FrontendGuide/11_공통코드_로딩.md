# 11. 공통코드 로딩

## 📋 이 단계의 목적

HTOMSS 프로젝트의 **공통코드 로딩 패턴**을 이해하고 올바르게 구현합니다.

---

## 1️⃣ 기본 패턴

### 1.1 useCommonCodeStore 사용

```typescript
import { useCommonCodeStore } from '@/stores/commonCodeStore';
import { computed } from 'vue';

const commonCodeStore = useCommonCodeStore();
```

### 1.2 onMounted에서 일괄 로드

```typescript
onMounted(async () => {
  // ⭐ Nexacro의 getComCodeComboSync 대응
  await commonCodeStore.loadMulti([
    { QueryId: 'BMCM010Query.ContCombo' },                           // 차수
    { QueryId: 'MNPS010Query.DateCond', customParam },               // 조회기간구분
    { QueryId: 'MNPS010Query.StatComboIn', customParam },            // 진행상태
    { QueryId: 'COM_PBL_0005', whereClause: "P_CODE_CD='C'" },       // 취소상태
    { QueryId: 'COM_PBL_0010' },                                     // IPG (그리드용)
    { QueryId: 'COM_PBL_0008' },                                     // 제작구분 (그리드용)
  ]);
});
```

### 1.3 computed로 옵션 정의

```typescript
// customParam 정의
const customParam = 'BUSN_SC_IN=A1 PROG_ID=PMDP010';

// computed로 옵션 가져오기
const contNoOptions = computed(() => 
  commonCodeStore.options('BMCM010Query.ContCombo', true)
);

const prgStatOptions = computed(() => 
  commonCodeStore.options('MNPS010Query.StatComboIn', true, '전체', '', undefined, customParam)
);

const cancStatOptions = computed(() => 
  commonCodeStore.options('COM_PBL_0005', true, '전체', '', "P_CODE_CD='C'")
);
```

**중요**:
- `true` = "전체" 옵션 추가
- `customParam` = HQML에서 사용하는 파라미터
- `whereClause` = 추가 필터 조건

## 2️⃣ 동적 로딩 패턴

### 2.1 Select 변경 이벤트

```typescript
// 차수 변경 시 부체계구분 동적 로드
const onContNoChange = (event: any) => {
  const contNo = event.value;
  if (contNo) {
    loadStrCdOptions(contNo);
  } else {
    strCdOptions.value = [];
  }
};
```

### 2.2 동적 로딩 함수

```typescript
const strCdOptions = ref<any[]>([]);

const loadStrCdOptions = async (contNo: string) => {
  if (!contNo) {
    strCdOptions.value = [];
    return;
  }
  
  // customParam에 파라미터 전달
  const param = `CONT_NO=${contNo}`;
  
  // 공통코드 로드
  await commonCodeStore.loadMulti([
    { QueryId: 'MNOD090Query.StrCombo', customParam: param }
  ]);
  
  // ⭐ CRITICAL: options 호출 시에도 동일한 customParam 전달 (캐시 키 매칭)
  strCdOptions.value = commonCodeStore.options(
    'MNOD090Query.StrCombo', 
    true,           // 전체 옵션 추가
    '전체',          // 전체 옵션 레이블
    '',             // 전체 옵션 값
    undefined,      // whereClause
    param           // customParam (캐시 키로 사용됨!)
  );
};
```

**중요**: 
- `loadMulti` 호출 시 `customParam`
- `options` 호출 시에도 **동일한 customParam** 전달해야 캐시에서 찾을 수 있음!

## 3️⃣ Select 컴포넌트 연결

```vue
<SearchFormField v-slot="{ props, invalid }" name="prg_stat">
  <SearchFormLabel>진행상태</SearchFormLabel>

      placeholder="선택"

  </SearchFormContent>
```

## 4️⃣ options() 메서드 파라미터

### 4.1 메서드 시그니처

```typescript
commonCodeStore.options(
  QueryId: string,           // 공통코드 쿼리 ID
  addAll: boolean = false,   // "전체" 옵션 추가 여부
  allLabel: string = '전체',  // "전체" 옵션 레이블
  allValue: string = '',     // "전체" 옵션 값
  whereClause?: string,      // WHERE 절 추가 조건
  customParam?: string       // 커스텀 파라미터 (캐시 키로 사용)
)
```

### 4.2 사용 예시

```typescript
// 1. 기본 사용 (전체 옵션 없음)
const ipgOptions = computed(() => 
  commonCodeStore.options('COM_PBL_0010')
);

// 2. 전체 옵션 포함
const prgStatOptions = computed(() => 
  commonCodeStore.options('MNPS010Query.StatComboIn', true)
);

// 3. customParam 사용
const customParam = 'BUSN_SC_IN=A1 PROG_ID=PMDP010';
const dateTypeOptions = computed(() => 
  commonCodeStore.options('MNPS010Query.DateCond', true, '전체', '', undefined, customParam)
);

// 4. whereClause 사용
const cancStatOptions = computed(() => 
  commonCodeStore.options('COM_PBL_0005', true, '전체', '', "P_CODE_CD='C'")
);

// 5. 동적 customParam (파라미터 포함)
const param = `CONT_NO=${contNo}`;
const strCdOptions = commonCodeStore.options('MNOD090Query.StrCombo', true, '전체', '', undefined, param);
```

## 6️⃣ 자주 사용하는 공통코드

### 6.1 PMDP010/PMDP020 공통

```typescript
const customParam = 'BUSN_SC_IN=A1 PROG_ID=PMDP010';

await commonCodeStore.loadMulti([
  { QueryId: 'BMCM010Query.ContCombo' },                       // 차수
  { QueryId: 'MNPS010Query.DateCond', customParam },           // 기간구분
  { QueryId: 'MNPS010Query.StatComboIn', customParam },        // 진행상태
  { QueryId: 'COM_PBL_0005', whereClause: "P_CODE_CD='C'" },   // 취소상태
  { QueryId: 'COM_PBL_0010' },                                 // IPG
  { QueryId: 'COM_PBL_0008' },                                 // 제작구분
]);
```

### 6.2 공통코드 테이블

| QueryId | 설명 | customParam | whereClause |
|---------|------|-------------|-------------|
| BMCM010Query.ContCombo | 차수 | - | - |
| MNOD090Query.StrCombo | 부체계구분 | CONT_NO={차수} | - |
| MNPS010Query.DateCond | 기간구분 | BUSN_SC_IN=A1 PROG_ID={화면ID} | - |
| MNPS010Query.StatComboIn | 진행상태 | BUSN_SC_IN=A1 PROG_ID={화면ID} | - |
| COM_PBL_0005 | 취소상태 | - | P_CODE_CD='C' |
| COM_PBL_0010 | IPG | - | - |
| COM_PBL_0008 | 제작구분 | - | - |
