# 컴포넌트 매핑표

## 📋 이 문서의 목적

Nexacro 컴포넌트를 Vue (PrimeVue) 컴포넌트로 변환할 때 참고하는 매핑 테이블입니다.

**참고 (PFY)**: PMDP010 등 예시 화면 소스는 `src/pages/sy/ds/` 하위를 기준으로 합니다.

---

## 1️⃣ 입력 컴포넌트

### Combo (드롭다운)

| Nexacro | Vue (PrimeVue) | 비고 |
|---------|----------------|------|
| `<Combo>` | `<Select>` | 단일 선택 드롭다운 |
| innerdataset | options (Array) | 데이터 소스 |
| codecolumn | optionValue | 값 필드 |
| datacolumn | optionLabel | 라벨 필드 |
| value | v-model | 양방향 바인딩 |

**Nexacro 예시**:
```xml
<Combo id="cbPRG_STAT"
  innerdataset="dsCodePrgStat"
  codecolumn="CODE"
  datacolumn="CODE_NM"
/>
```

**Vue 변환**:
```vue
<Select
  v-model="form.prg_stat"
  :options="prgStatOptions"
  optionValue="CODE"
  optionLabel="CODE_NM"
  placeholder="전체"
/>
```

### Edit (텍스트 입력)

| Nexacro | Vue (PrimeVue) | 비고 |
|---------|----------------|------|
| `<Edit>` | `<InputText>` | 한 줄 텍스트 입력 |
| value | v-model | 양방향 바인딩 |
| maxlength | maxlength | 최대 길이 |
| readonly | readonly | 읽기 전용 |

**Nexacro 예시**:
```xml

/>
```

**Vue 변환**:
```vue
<InputText
  v-model="form.cont_no"
  maxlength="20"
  placeholder="계약번호 입력"
/>
```

### Calendar (날짜 선택)

| Nexacro | Vue (PrimeVue) | 비고 |
|---------|----------------|------|
| `<Calendar>` | `<Calendar>` | 날짜 선택 |
| value | v-model | 양방향 바인딩 (Date 객체) |
| dateformat | dateFormat | 표시 포맷 |

**Nexacro 예시**:

### Radio (라디오 버튼)

| Nexacro | Vue (PrimeVue) | 비고 |
|---------|----------------|------|
| `<Radio>` | `<RadioButton>` | 단일 라디오 버튼 |
| - | `<RadioButtonGroup>` | 라디오 버튼 그룹 (커스텀) |
| value | v-model | 양방향 바인딩 |
| codecolumn | value | 선택 값 |

**Nexacro 예시**:
```xml
<Radio id="rdoASK_DT" value="askDt" />
<Radio id="rdoREQ_DT" value="reqDt" />
```

**Vue 변환**:
```vue
<!-- RadioButtonGroup 사용 (SearchForm) -->
{
  name: 'searchDtType',
  component: 'RadioButtonGroup',
  props: {
    options: [
      { label: '의뢰일자', value: 'askDt' },
      { label: '요청일자', value: 'reqDt' },
    ],
  },
}

<!-- 직접 사용 -->
<div class="radio-group">
  <RadioButton
    v-model="form.searchDtType"
    value="askDt"
    inputId="askDt"
  />
  <label for="askDt">의뢰일자</label>
  
  <RadioButton
    v-model="form.searchDtType"
    value="reqDt"
    inputId="reqDt"
  />
  <label for="reqDt">요청일자</label>
</div>
```

## 3️⃣ 그리드 컴포넌트

### Grid

| Nexacro | Vue (PrimeVue) | 비고 |
|---------|----------------|------|
| `<Grid>` | `<DataTable>` | 데이터 그리드 |
| binddataset | :value | 데이터 소스 |
| oncellclick | @row-click | 행 클릭 |
| autofittype | - | Vue에서는 수동 설정 |

**Nexacro 예시**:
```xml
<Grid id="grdMain"
  binddataset="dsMain"
  autofittype="col"
>
  <Formats>
    <Format id="default">
      <Columns>
        <Column size="80"/>
        <Column size="120"/>
      </Columns>
      <Rows>
        <Row size="24" band="head"/>
        <Row size="24"/>
      </Rows>
      <Band id="head">
        <Cell text="선택"/>
        <Cell col="1" text="계약번호"/>
      </Band>
      <Band id="body">
        <Cell displaytype="checkboxcontrol" edittype="checkbox"/>
        <Cell col="1" text="bind:CONT_NO"/>
      </Band>
    </Format>
  </Formats>
</Grid>
```

**Vue 변환**:
```vue
<DataTable
  v-model:selection="selectedRows"
  :value="fetchedMainData"
  scrollable
  scrollHeight="540px"
  :virtualScrollerOptions="{ itemSize: 46 }"
  selectionMode="multiple"
  dataKey="CONT_NO"
>
  <Column
    selectionMode="multiple"
    headerStyle="width: 3rem"
  />
  <Column
    field="CONT_NO"
    header="계약번호"
    style="min-width: 120px"
  />
</DataTable>
```

### Grid Column 타입 매핑

#### 체크박스

**Nexacro**:
```xml
<Cell displaytype="checkboxcontrol" edittype="checkbox"/>
```

**Vue**:
```vue
<Column selectionMode="multiple" />
```

#### 텍스트

**Nexacro**:
```xml
<Cell text="bind:CONT_NO"/>
```

**Vue**:
```vue
<Column field="CONT_NO" header="계약번호" />
```

#### 드롭다운

**Nexacro**:
```xml
<Cell displaytype="combocontrol" edittype="combo"
  combodataset="dsCodePrgStat"
  combocodecol="CODE"
  combodatacol="CODE_NM"
/>
```

**Vue**:
```vue
<Column field="PRG_STAT" header="진행상태">
  <template #body="{ data }">
    {{ getCodeName(data.PRG_STAT, prgStatOptions) }}
  </template>
</Column>
```

#### 템플릿 (커스텀)

**Nexacro**:
```xml
<Cell text="expr:ASK_DT.substr(0,4)+'-'+ASK_DT.substr(4,2)+'-'+ASK_DT.substr(6,2)"/>
```

**Vue (권장하지 않음 - 성능 이슈)**:
```vue
<Column field="ASK_DT" header="의뢰일자">
  <template #body="{ data }">
    {{ formatDate(data.ASK_DT, 'YYYYMMDD', 'YYYY-MM-DD') }}
  </template>
</Column>
```

**Vue (권장 - 전처리)**:
```typescript
watch(fetchedMainData, async (newData) => {
  await nextTick();
  newData.forEach(row => {
    row.ASK_DT_FORMATTED = formatDate(row.ASK_DT, 'YYYYMMDD', 'YYYY-MM-DD');
  });
});
```

```vue
<Column field="ASK_DT_FORMATTED" header="의뢰일자" />
```

## 5️⃣ 데이터 컴포넌트

### Dataset

| Nexacro | Vue | 비고 |
|---------|-----|------|
| `<Dataset>` | ref<Array> | 반응형 배열 |
| getColumn() | data[index].field | 객체 접근 |
| setColumn() | data[index].field = value | 직접 할당 |
| addRow() | data.push({}) | 배열 메서드 |
| deleteRow() | data.splice(index, 1) | 배열 메서드 |

**Nexacro 예시**:
```javascript
this.dsMain.addRow();
this.dsMain.setColumn(0, "CONT_NO", "1234");
var contNo = this.dsMain.getColumn(0, "CONT_NO");
```

**Vue 변환**:
```typescript
const mainData = ref<MainData[]>([]);

// addRow
mainData.value.push({
  CONT_NO: '',
  // ...
});

// setColumn
mainData.value[0].CONT_NO = '1234';

// getColumn
const contNo = mainData.value[0].CONT_NO;
```

### 메시지 표시

**Nexacro**:
```javascript
this.gfnAlert("저장되었습니다.");
this.gfnConfirm("삭제하시겠습니까?");
```

**Vue**:
```typescript
import { useToast } from 'primevue/usetoast';
import { useConfirm } from 'primevue/useconfirm';

const toast = useToast();
const confirm = useConfirm();

// Alert
toast.add({
  severity: 'success',
  summary: '성공',
  detail: '저장되었습니다.',
  life: 3000,
});

// Confirm
confirm.require({
  message: '삭제하시겠습니까?',
  header: '확인',
  icon: 'pi pi-exclamation-triangle',
  accept: () => {
    // 삭제 로직
  },
});
```

## 📊 전체 매핑 요약
