# 12. 커스텀 컴포넌트 가이드

## 📋 개요

`pfy-front/src/components/common/` 폴더에는 PrimeVue 컴포넌트를 프로젝트 요구사항에 맞게 커스터마이징한 래퍼 컴포넌트들이 있습니다.

**중요**: 항상 PrimeVue를 직접 import하지 말고, 이 커스텀 컴포넌트들을 사용하세요!

---

## 🔍 Import 패턴 결정 규칙

### Step 1: index.ts 확인
```bash
# 컴포넌트 폴더에 index.ts가 있는지 확인
ls /components/common/[componentName]/index.ts
```

### Step 2: Import 방식 결정
- ✅ **index.ts 있음** → Destructuring import
- ✅ **index.ts 없음** → Direct .vue import

### Step 3: spov010 참조 확인
```bash
# 실제 사용 패턴 확인
grep "import.*ComponentName" src/pages/sy/ds/spov010/**/*.vue
```

### 🔹 Button
**경로**: `@/components/common/button`
**index.ts**: ✅ 있음

#### Import 패턴
```typescript
// ✅ CORRECT
import { Button, ToggleButton } from '@/components/common/button';

// ❌ WRONG
import Button from 'primevue/button';
import Button from '@/components/common/button/Button.vue';
```

#### 사용법
```vue
<template>
  <!-- 기본 버튼 -->
  <Button label="조회" icon="pi pi-search" @click="handleSearch" />
  
  <!-- 아웃라인 버튼 -->
  <Button 
    label="삭제" 
    severity="danger" 
    variant="outlined"
    @click="handleDelete" 
  />
  
  <!-- 아이콘만 -->
  <Button 
    icon="pi pi-plus" 
    severity="success"
    rounded
    @click="handleAdd" 
  />
  
  <!-- 토글 버튼 -->
  <ToggleButton 
    v-model="checked" 
    onLabel="ON" 
    offLabel="OFF" 
  />
</template>
```

#### Props
- `label`: string - 버튼 텍스트
- `icon`: string - PrimeIcons 클래스명
- `severity`: 'secondary' | 'success' | 'info' | 'warn' | 'danger' | 'contrast'
- `variant`: 'text' | 'outlined' - 버튼 스타일
- `rounded`: boolean - 둥근 버튼
- `loading`: boolean - 로딩 스피너 표시
- `disabled`: boolean - 비활성화

#### 사용 시나리오
- SearchForm의 조회/초기화 버튼
- DataTable의 액션 버튼
- Dialog의 확인/취소 버튼
- SumGrid의 상태별 필터 버튼

  confirm.require({
    message: '정말 삭제하시겠습니까?',
    header: '삭제 확인',

    extraActions: [
      {
        label: '임시저장',
        onClick: () => {
          console.log('임시저장');
        }
      }

**사용법**: PrimeVue의 `useToast` 훅 사용

**⚠️ CRITICAL**: `alert()` 대신 항상 Toast를 사용하세요!

#### 기본 사용법
```vue
<script setup lang="ts">
import { useToast } from 'primevue/usetoast';

const toast = useToast();

const showSuccess = () => {
  toast.add({
    severity: 'success',
    summary: '성공',
    detail: '저장되었습니다',
    life: 3000
  });
};

const showError = () => {
  toast.add({
    severity: 'error',
    summary: '오류',
    detail: '오류가 발생했습니다',
    life: 3000
  });
};

const showInfo = () => {
  toast.add({
    severity: 'info',
    summary: '정보',
    detail: '추가 정보입니다',
    life: 3000
  });
};

const showWarn = () => {
  toast.add({
    severity: 'warn',
    summary: '경고',
    detail: '주의가 필요합니다',
    life: 3000
  });
};
</script>
```

#### Toast Options
- `severity`: 'success' | 'info' | 'warn' | 'error' - 메시지 유형
- `summary`: string - 제목
- `detail`: string - 상세 메시지
- `life`: number - 자동 닫힘 시간 (ms, 기본: 3000)
- `closable`: boolean - 닫기 버튼 표시 (기본: true)

#### 실전 예시

**✅ 검증 메시지**
```typescript

```

#### Props
- `value`: any[] - 테이블 데이터

- `scrollHeight`: string - 스크롤 높이

- `selectionMode`: 'single' | 'multiple' - 선택 모드
- `selection`: any | any[] - 선택된 행 (v-model)
- `paginator`: boolean - 페이지네이션
- `rows`: number - 페이지당 행 수
- `sortField`: string - 기본 정렬 필드
- `sortOrder`: 1 | -1 - 정렬 순서

#### Column Props
- `field`: string - 데이터 필드명
- `header`: string - 헤더 텍스트
- `style`: string - 스타일
- `sortable`: boolean - 정렬 가능
- `frozen`: boolean - 고정 컬럼
- `alignFrozen`: 'left' | 'right' - 고정 위치

#### Events
- `row-click`: 행 클릭
- `row-dblclick`: 행 더블클릭
- `row-select`: 행 선택
- `row-unselect`: 행 선택 해제
