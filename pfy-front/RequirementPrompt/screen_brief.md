## 2. 화면 유형 (Template 선택)

> 아래 중 하나를 선택하세요. 모르면 비워두면 AI가 자동 선택합니다.

- [ ] **TypeA** — 검색 + 목록 조회 + 상세 팝업 (Dialog)
- [x] **TypeA** — 검색 + 목록 조회 + 상세 팝업 ← **이 화면에 해당**
- [ ] **TypeB** — 입력/수정 단독 폼 페이지
- [ ] **TypeC** — 좌측 트리 + 우측 데이터 그리드
- [ ] **TypeD** — 대시보드 (집계 카드 + 요약 테이블)

## 8. 상태값 정의

> DotStatusText, Tag 등에 사용할 상태값과 색상을 정의합니다.

| 코드값 | 표시 텍스트 | 색상/severity |
| :--- | :--- | :--- |
| RECEIVED | 접수 | `info` (파란색) |
| IN_PROGRESS | 처리중 | `warning` (주황색) |
| COMPLETED | 처리완료 | `success` (초록색) |
| REJECTED | 기각 | `danger` (빨간색) |
